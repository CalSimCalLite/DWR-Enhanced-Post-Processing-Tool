Retrieving DTS:
   When you have a DTS table opened, click on Table|Retrieve menu item, the derived time series in the specified dss file will be displayed.

Another way to retrieve a DTS is, click on DTS menu, highlight Retrieve sub menu, you will see all the DTS in current project are listed, click on any DTS, its data will be displayed.

Also see Listing DTS.

Note: If you don't have a dss file opened for retrieving data, then an exception will be displayed.
