5. Tutorial

At startup, the Run Settings dashboard of the CalLite GUI is visible. The tabs with white backgrounds and blue letters relate to model results, while the remaining dashboards for model input and control have gray tabs and black letters. Figure 5-1 shows the Run Settings dashboard as well as the various tabs. The following sub-sections provide a simple example application of the CalLite model. For this example, no changes to the Run Settings dashboard will be pursued.
