3.1. Project Configuration

The Project Configuration’s main components are the Scenarios and Display panels. The Project Name and Description are also defined in this window. The panels on this window are as follows:

 

Scenarios: Here, the user can add or delete scenarios to determine which scenario results will be displayed. The two scenarios  and type of results to display (either for a single base scenario, a comparison of scenarios, or the difference between scenarios) can also be selected. The clear or clear all button can be used to clear selected scenarios. For difference plots, the base selection button sets which scenario to use as the basis for comparison when computing difference values.

 

Display: Options for displaying results:

  • Start and end month and year
  • Flow data in either CFS or TAF (thousand-acre feet). Variables other than flow have fixed units (TAF for storage, μmhos/cm for salinity). Note: μmhos is a unit of specific conductivity.
  • Monthly time-series plot
  • Exceedance plots and Box and Whiskers plots. These plots can be shown for all monthly values or annual totals. The annual total exceedance plots are only shown for flow variables and the units are always TAF.
  • Table of monthly values. For flow variables, the annual total shown is always in TAF.
  • Summary table. The summary table can show five different statistics (average, maximum, minimum, standard deviation, and median) computed from monthly and annual values, summarized by four different water year indices, and for the three driest periods in 1922-2003. When summarizing data by water year type, the Sacramento Valley 40-30-30 Index, Feather Index, and SJR (San Joaquin Valley 60-20-20) Index use October as the first month of the water year; the Shasta Index uses March.
