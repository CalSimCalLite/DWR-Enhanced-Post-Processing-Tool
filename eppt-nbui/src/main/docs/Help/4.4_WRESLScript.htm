4.4. Running the WRESL Script

Running the WRESL script can be performed in two ways: Clicking (1) the Tools dropdown option (Figure 4-24) or (2) The WRESL Script main menu icon (Figure 4-25).

Figure 4-24: Tools > Run WRESL Script.

 

Figure 4-25: WRESL Script icon.

‎

The Run WRESL Script window will appear and show the loaded Scenarios. Once a Scenario is chosen and the Scenarios panel is filled out correctly, click “Run WRESL.”

 

 

Figure 4-26: Run WRESL Script window.

 

After running the WRESL Script, the QA/QC Report can be generated.
