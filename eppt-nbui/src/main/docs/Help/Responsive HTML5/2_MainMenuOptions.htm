2. Main Menu Options

The EPPT Application relies on the NetBeans Platform’s window management. Subsections of windows are referred to as panels.

This segregates the program into distinct windows. The following windows are currently part of the EPPT application:

 

  • Project Configuration
  • Quick Results
  • Custom Results
  • Data Analysis 

 

Most user choices in the EPPT are handled on a series of windows (Figure 2-1) accessed through tabs across the top of the interface. One window specifies scenario and display settings (Project Configuration) while three windows view results: Quick Results, Custom Results, Data Analysis.

 

EPPT includes six dropdown menus:

 

  • File: Options to load, run, save, save as, or delete an existing scenario.
  • View: Enables IDE log view, full screen view
  • Navigate: Return to previous window
  • Tools: Runs WRESL Script, generates QA/QC report, opens HEC-DSSVue, or opens EPPT Options Menu
  • Window:  Navigate and close windows or documents
  • Help: Access to the help documentation and the current EPPT version information.
