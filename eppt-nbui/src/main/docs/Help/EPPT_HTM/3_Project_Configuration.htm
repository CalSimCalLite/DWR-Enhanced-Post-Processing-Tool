3. Project Configuration

 

Project Configurations are a new feature to the EPPT, not available in CalLite. Project Configurations are a composition of the available Scenarios and display options. (See Figure 3-1) The EPPT allows the user to save these configurations out to a JSON formatted file. The user can then reload the saved configuration to continue operations with the same selections. When the EPPT is opened, the last loaded configuration will be restored.

 

The New Project Configuration File menu item and Toolbar button allow the user to define a new Project Configuration. The user specifies a name, description, and file path to store the configuration options. The Load Project Configuration File menu item and Toolbar button load a configuration from the selected file. The Save Project Configuration File menu item and Toolbar button save the current options to the active Project Configuration file.

 

The EPPT Project Configuration files will specify the Scenario DSS Files relative to the EPPT Project Configuration directory unless they are in a separate folder structure. Then the absolute path will be saved instead.

 

The Project Configuration format is under revision and will see increased functionality in the following versions of EPPT.

 

Scenarios. Here the user can add or delete scenarios to determine which scenario results will be displayed. The number of scenarios and type of results to display (either for a single base scenario, a comparison of scenarios, or the difference between scenarios) can also be selected. The radio button next to each scenario name can be used to select which scenario to delete. For difference plots, the radio button sets which scenario to use as the basis for comparison when computing difference values.

 

Display. Options for displaying results:

  • Start and end month and year.
  • Flow data in either CFS or TAF (thousand-acre feet). Variables other than flow have fixed units (TAF for storage, μmhos/cm for salinity). Note: μmhos is a unit of specific conductivity.
  • Monthly time-series plot.
  • Exceedance plots and Box and Whiskers plots. These plots can be shown for individual months, all monthly values, or annual totals. The annual total exceedance plots are only shown for flow variables and the units are always TAF.
  • Table of monthly values. For flow variables, the annual total shown is always in TAF.
  • Summary table. The summary table can show five different statistics (average, maximum, minimum, standard deviation, and median) computed from monthly and annual values, summarized by four different water year types, and for the three driest periods in 1922-2003. When summarizing data by water year type, the Sacramento Valley 40-30-30 Index, Feather Index, and SJR (San Joaquin Valley 60-20-20) Index use October as the first month of the water year; the Shasta Index uses March.
