2. Main Menu Options

Most user choices in the EPPT are handled on a series of dashboards accessed through tabs across the top of the interface. One dashboard shows scenario settings and display options (Project Configuration) while three dashboards view results : (Quick Results, Custom Results, Data Analysis). See Figure 2-1.

 

There are six dropdown menus:

File: Options to load, run, save, save as, or delete an existing scenario.

View: Enables IDE log view, full-screen view

Navigate: Return to the previous dashboard

Tools: Opens HEC-DSSVue or EPPT Options Menu

Dashboards: Navigate and close dashboards or documents

Help: Access to the help system and the current EPPT version information.
