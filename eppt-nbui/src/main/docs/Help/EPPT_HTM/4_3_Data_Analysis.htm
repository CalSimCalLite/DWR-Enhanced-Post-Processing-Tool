4.3. Data Analysis

 

This dashboard allows the user to produce a standardized comparison or corroboration report comparing two CalLite scenarios, two CalSim II scenarios, two CalSim 3 scenarios or between scenarios of any two of the previously mentioned models.

 

The EPPT produces this report by calling one of the CalLite utilities, the CalLite report tool. The report shows storage in major reservoirs, flows at key river segments, CVP and SWP deliveries, and other scenario outputs.

 

In order to generate the report, use the Select File buttons to select a report template file (.inp file), two DSS output files, and a report output file name. Next, enter names for each of the two studies to be compared (defaults are set to CalLite 1 and CalLite 2), and finally hit the Generate Report button. Optional information such as notes, assumptions, and modeler name can also be added to the report.

 

  • CalLite-CalLite comparison:
    • The default report template file (callite_scenario_comparison.inp) will compare two CalLite scenarios.
  • CalSim II-CalSim II comparison:
    • Click “Select File …” in the “Report Template File” row and select “calsim_calsim_comparison.inp”.
  • CalSim 3-CalSim 3 comparison:
    • Click “Select File …” in the “Report Template File” row and select “calsim3_calsim3_comparison.inp”.
  • CalSim II-CalLite corroboration:
    • The report tool can also be used to compare CalLite results to CalSim results. To do this, the DSS Result File #1 must be set to the output DSS from a CalSim simulation, the DSS Result File #2 must be set to the output DSS from a CalLite simulation and the report template file must be set to calsim_callite_corroboration.inp.  
  • CalSim 3-CalLite corroboration:
    • To compare CalSim 3 results to CalLite results, the DSS Result File #1 must be set to the output DSS from a CalSim 3 simulation, the DSS Result File #2 must be set to the output DSS from a CalLite simulation and the report template file must be set to calsim3_callite_corroboration.inp.
  • CalSim 3-CalSim II corroboration:
    • To compare CalSim 3 results to CalSim II results, the DSS Result File #1 must be set to the output DSS from a CalSim 3 simulation, the DSS Result File #2 must be set to the output DSS from a CalSim II simulation and the report template file must be set to calsim3_calsim_corroboration.inp.

 

For more detailed description of the report tool, and instructions on how to edit the report template files, see Appendix x in the EPPT Reference Manual.
