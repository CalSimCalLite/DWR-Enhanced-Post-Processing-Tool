4.3. Generating Data Analysis PDF

Data Analysis report templates for generating convenient sets of tables and charts in PDF format are accessible from the Data Analysis window. For this example, DCR2015 and DCR2017 scenario result files were used with the “calsim_calsim_scenario_comparison.inp” template. Figures 4-22 and 4-23 provide an example of the PDF generation process and a report page sample.

 

Figure 4-22. The EPPT allows rapid generation of report-quality charts and tables from different sets of model results.

 

Click “Generate Report.” The default file title is “report.pdf.” Alternative titles may be used. If that file name already exists, a warning message asking to overwrite will appear. Click “Yes” regardless to create the Data Analysis PDF Report (Figure 4-23).

 

Figure 4-23. A sample of a EPPT Report.
