List<DataTableModel> gov.ca.water.calgui.bus_delegate.impl.ScenarioDeleImp.getScenarioTableData ( List< String >  fileNames )

This will add the Current_Scenario.CLS file to the file names and then build the List of DataTableModel.

Parameters
fileNames File names that should be displayed in the ScenarioFrame. Send null if you want to see only the current Scenario in the frame.
Returns
Will return only one object if the file name is null that is the Current Scenario. Otherwise it will return three objects in which the 1st is Current Scenario, 2nd is the comparison and the 3ed is differences.
